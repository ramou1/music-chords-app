# Minha Despensa 
Aplicativo de cifras de músicas desenvolvido em IONIC 5.

## O que o app faz?

:memo: Veja o perfil dos artistas com as correspondentes cifras; <br />
:memo: Veja a letra da música e/ou cifra disponível de cada artista; <br />
